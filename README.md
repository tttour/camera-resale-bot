# カメラ転売リサーチボット 📸

カメラのキタムラ中古品とメルカリ売り切れ価格を自動比較し、利益が出る商品をWindows通知で教えてくれるツールです。

---

## 🔄 動作フロー

```
① キタムラ売れ筋カメラ TOP10 を取得
    ↓
② 各商品の「同型商品」を安い順で取得
    ↓
③ メルカリで同型番の売り切れ価格を取得（中央値・平均）
    ↓
④ 利益計算 → 閾値超えたらWindows通知🔔
```

---

## ⏰ 実行間隔について

| 更新頻度 | 推奨設定 |
|---------|---------|
| キタムラ在庫（数時間〜1日） | **6時間ごと** |
| メルカリ売り切れ（数日）   | **6時間ごと** |

→ `config.json` の `interval_hours` で変更可能（デフォルト: 6）

---

## 🚀 起動方法

### テスト実行（最初はこちら）
```
run_test.bat をダブルクリック
```
ブラウザが表示された状態で2件のみ処理します。動作確認に使ってください。

### 通常実行（1回だけ）
```bash
python main.py
```

### 定期実行（6時間ごと自動）
```
run_bot.bat をダブルクリック
```
または：
```bash
python main.py --schedule
```

### コマンドラインオプション
| オプション | 説明 |
|-----------|------|
| `--schedule` | 6時間ごとに繰り返し実行 |
| `--test` | 2件のみ処理（動作確認用） |
| `--no-headless` | ブラウザを表示して実行 |
| `--config path` | 設定ファイルを指定 |

---

## ⚙️ 設定（config.json）

```json
{
  "settings": {
    "interval_hours": 6,          // 実行間隔（時間）
    "min_profit_yen": 3000,       // 通知する最低利益額（円）
    "mercari_fee_rate": 0.1,      // メルカリ手数料 10%
    "top_cameras_count": 10,      // チェックする売れ筋件数
    "same_type_max_pages": 3,     // 同型商品の最大取得ページ数
    "headless": true              // false にするとブラウザ表示
  }
}
```

## 💰 利益計算ロジック

```
買取価格（キタムラ）
  → 同型品の最安値

メルカリ売却価格
  = 売り切れ中央値 × 90%  ← 値引き交渉を考慮した安全マージン

純利益 = 売却価格 - メルカリ手数料(10%) - 送料(¥1,000) - 買取価格
```

## ⚠️ 注意事項

- **メルカリのスクレイピングは規約上グレーゾーン**です。過度なアクセスは避け、`request_delay_seconds` の設定を下げないでください（デフォルト: 2秒）。
- キタムラのサイト構造が変わった場合はセレクタの修正が必要になることがあります。
- ヘッドレスモードでブロックされる場合は `headless: false` に変更してください。

## 🛠️ Windows タスクスケジューラで自動起動

PCの起動時に自動でボットを起動したい場合:

1. `Win + R` → `taskschd.msc`
2. 「タスクの作成」
3. トリガー: ログオン時
4. 操作: `run_bot.bat` のフルパスを指定
